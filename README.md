## Installation

### 1. Clone the Repository
```bash
git clone <repo-url>
cd ai_hackathon_blackbuds
```

### 2. Create a Virtual Environment
```bash
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate
```
